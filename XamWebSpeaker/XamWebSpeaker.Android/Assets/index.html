侃侃朗读——可选段的网页朗读神器

朗读软件很多，能方便快捷的读公众号、读新闻客户端的却很少，本APP解决了这些痛点

  1. 保持原排版的选段朗读功能，可通过点击跳过前后的广告
  2. 自动滚屏（不支持自动翻页），解决系统内置朗读工具只读当前界面的弊端
  3. 离线语音合成，不另耗流量

安装 语音合成(TTS) 引擎

可以通过应用商店安装TTS引擎，目前推荐的中文TTS引擎是科大讯飞的【讯飞语记】，讯飞语记可以选不同的发音人。

设置默认引擎

设置首选引擎为【讯飞语记】，设置成其它不支持中文的引擎时，遇到中文时会无声音。部分机型可能出现【科大讯飞语音引擎】和【讯飞语记】两个选项，请选择【讯飞语记】

由于不同厂家定制ROM的不同，如果跳转失败，请手工到【设置->语言和输入->文字转语音输出】中进行设置

1. 使用帮助

界面分为三大区

  • 输入：浏览器的地址栏
    1. 粘贴：剪贴板内容填写到输入框
    2. 输入框：网址或文本
    3. 输入框中是文本：标记内容应当做为文本进行显示
    4. 浏览：跳转到浏览器，访问输入框中的网址或显示文本
  • 朗读：浏览器界面
    1. 开始：开始朗读
    2. 暂停：暂停朗读
  • 设置
    1. 文本转语音：可设置语速、音量与音调
    2. 语速：Android范围(0-2),推荐1.7；iPhone范围(0-1),推荐0.6
    3. 音调：Android部分TTS引擎不支持
    4. 屏幕常亮：可边听边看，主要用于图文混排网页

打开网页后，点击“开始”，即播放当前网页，可通过触屏点击快速跳转到其它段落

2. 使用方法

由于iOS与Android操作系统的区别，Android版使用起来更为便捷

2.1 打开网页(Android/iPhone通用)

设置网址或文本（输入界面）：可输入网址，也可以输入文本。如果是网址，请编辑成http开头的地址，点击“浏览”，反之开启“输入框中是文本”后再点“浏览”


复制链接（其它APP）：iPhone版大部分APP只能选择使用复制链接功能

复制文本（任意APP）：部分网页无法朗读，如使用iframe技术的网页无法跨域。解决的办法是网页中长按，弹出上下文菜单后全选复制，粘贴到输入框。如：百度移动版网页

2.2 打开网页(Android版)

在浏览器打开（其它APP）：在其它软件的中“在浏览器打开”中选择本APP。如：微信

发送到（其它APP）：在其它软件的“分享”中选择“神器”，在弹出的对话框中去除多余的文字，只保留网址，点击“网址浏览”。如：网易新闻客户端

分享至邮件（其它APP）：部分软件只有分享到微信或邮件的功能，可以选择邮件后，再选本APP，在弹出的对话框后，点击“文本显示”(部分APP邮件分享只会提供摘要)。如：万得APP
